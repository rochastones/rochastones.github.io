# rochastones.github.io
Website Memoria Afro-brasileira

O projeto 'Memória Afro-Brasileira' é um portal educacional desenvolvido com o objetivo de divulgar e preservar a história, cultura e contribuições afro-brasileiras no contexto nacional. O site integra recursos didáticos, acervos digitais e informações históricas com acessibilidade e usabilidade modernas.

1. Resumo 
O portal tem finalidade cultural e educacional, servindo como repositório de informações, vídeos e materiais didáticos que incentivam o ensino da história afro-brasileira, conforme previsto pela Lei 10.639/2003. O projeto foi elaborado como parte de um trabalho acadêmico no IFSP, unindo fundamentos de desenvolvimento web e valorização da herança afro-brasileira.

2. Arquitetura do Sistema
A estrutura do projeto é composta por cinco páginas principais e um arquivo de mapa do site escrito em xml, que lista todas as URLs existentes.
- index.html: Página inicial do portal.
- sobre.html: Apresenta o contexto histórico e a linha do tempo de marcos culturais.
- acervos.html: Lista museus e instituições relevantes.
- educacao.html: Reúne vídeos, guias e materiais educativos.
- contato.html: Disponibiliza formulário para interação com o público.
- sitemap.xml: Ajuda o mecanismo de busca a restrear e indexar o conteúdo. 
Os arquivos estão interligados por um menu de navegação unificado e utilizam o framework Bootstrap 5.3.2 para responsividade e estilo.

3. Design e Usabilidade
O layout foi planejado com base em princípios de clareza, contraste e responsividade. As cores remetem à identidade afro-brasileira, com predominância de tons terrosos e dourados. O Bootstrap garante adaptação a dispositivos móveis. Todas as imagens incluem textos alternativos (atributo alt) para acessibilidade.

4. Tecnologias Utilizadas
- HTML5 para estrutura semântica;
- CSS para estilização e layout;
- Bootstrap 5.3.2 para componentes responsivos;
- JavaScript para validação de formulários e interatividade;
- CDN externas para otimização de carregamento;
- Schema.org (JSON-LD) para metadados e SEO;
- Links externos com políticas de segurança (rel='noopener').
